Algorithm Performance

Exact Algorithm Results

  • Tested on multiple datasets.
  • Correctly identified densest subgraphs.
  • Longer execution time, especially for large graphs.

CoreExact Algorithm Results

  • Faster execution due to core decomposition.
  • Scalable to large graphs with dense regions.
  • Equally accurate in identifying densest subgraphs.

Summary of Key Findings

  • CoreExact significantly reduces execution time.
  • Both algorithms validate densest subgraphs accurately.
  • Core decomposition is crucial for practical scaling.
