Computing Dense Subgraphs Using Network Flow

Efficient techniques for finding densest subgraphs in graphs using network flow-based algorithms.

Project Overview

Finding densest subgraphs is a critical problem in graph theory and network analysis, with applications including:

  • Social network analysis (detecting communities)
  • Bioinformatics (protein interaction networks)
  • Recommendation systems (grouping similar users/items)
  • Fraud detection (identifying dense collusion groups)

A densest subgraph maximizes the density function — typically the ratio of edges to vertices — in a subset of a given graph.

Problem Statement

Given a graph G(V, E) where:

  • V = Set of vertices
  • E = Set of edges

The objective is to find a subgraph D ⊆ G that maximizes edge density using efficient flow-based algorithms.

Algorithms Implemented

We implemented and compared two approaches:

  • Exact Algorithm using binary search and flow networks.
  • CoreExact Algorithm utilizing (k,Ψ)-core decomposition for faster computation.
