Computing Dense Subgraphs Using Network Flow

Efficient techniques for finding densest subgraphs in graphs using network flow-based algorithms.

1. Introduction

This report discusses the key ideas and findings from the paper "Efficient Algorithms for Densest Subgraph Discovery" by Yixiang Fang and collaborators. The paper addresses the Densest Subgraph Discovery (DSD) problem, aiming to extract subgraphs with maximum density based on edge count or clique structures.

DSD appears across domains like social network analysis, biological systems, and database indexing. Traditional methods struggle with computational efficiency on large graphs.

To address this, the paper proposes innovative techniques using core decomposition, which significantly reduce search space and improve runtime performance. These algorithms demonstrate substantial speedups over previous methods.

2. Problem Statement

Given a graph G(V, E) where:

  • V = Set of vertices
  • E = Set of edges

The objective is to find a subgraph D ⊆ G that maximizes edge density using efficient flow-based algorithms.

3. Algorithms Implemented

We implemented and compared two approaches:

  • Exact Algorithm using binary search and flow networks.
  • CoreExact Algorithm using (k,Ψ)-core decomposition for optimization.
