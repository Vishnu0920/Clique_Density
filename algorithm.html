Algorithms Implemented

1) Exact Algorithm

Source: "Efficient Algorithms for Densest Subgraph Discovery" (Fang et al., 2019)

Key Idea: Perform a binary search over density values. At each step, construct a flow network and use minimum s-t cut to validate feasible subgraphs.

Steps:

  • Binary search on density values.
  • Build a flow network for each candidate density.
  • Use min s-t cut to check subgraph feasibility.

Time Complexity: O(n⋅(d−1 choose h−1) + (n|Λ| + min(n,|Λ|)^3) log n)

2) CoreExact Algorithm

Source: Same paper, optimized version.

Key Idea: Improve the Exact algorithm by performing (k,Ψ)-core decomposition to restrict the flow network only to dense regions.

Advantages:

  • Reduces graph size before binary search.
  • Significantly faster in practice.
  • Especially beneficial for large graphs with high clustering.

Time Complexity: Similar worst-case to Exact, but faster in practice.
